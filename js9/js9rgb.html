JS9 Demo: RGB composites

If you have three images of the same size, scale, and pointing direction (e.g. three energy cuts of a Chandra data set), you can display each one separately and assign to them one of the red, green, or blue colormaps. If you then select the menu option Color->rgbMode, they will be displayed as one RGB composite image. The current image (as shown in the File menu) determines which of the three independent colormaps is changed by contrast/bias mouse manipulation. To remove an image from the RGB composite, give it a different colormap. Re-join (or add a new image) by assigning one of the RGB colormaps. Turn off rgbMode to display the images separately.
