General information

All queries should select the ob (observing block) column, since the webpage needs it in order to display the buttons.

Every field is inspected by two people, and thus is in the database twice. Queries that show all fields with a comment submitted by both of the persons might not display properly. Those fields would be returned twice and the webpage can have each field at most once. Therefore it is preferable to query with SELECT DISTINCT.

Useful queries:

  • See all of the fields available: SELECT DISTINCT ob FROM kids_theliqc

  • Fields within sky coordinates (declination column is named de): SELECT DISTINCT ob FROM kids_theliqc WHERE ra>180 AND ra<220 AND de<0 AND de>-30

  • All fields with fringing: SELECT DISTINCT ob FROM kids_theliqc WHERE i_fringing = 1 ORDER BY ra, de ASC

  • The fields passed or ready to pass on to Edinburgh: SELECT DISTINCT ob FROM kids_theliqc WHERE submit_to_edinburgh = 1 ORDER BY ra, de ASC
